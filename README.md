# Thermostat

#### A Single Page Application to control a thermostat

## How to run:

1. In root, run:
- $ npm install
- You may need to also run $ npm install esbuild

2. In thermostat-backend/, run:
- $ node index.js

3. In thermostat-client-tc/, run:
- $ npm run build
- Open index.html and interact with the SPA
